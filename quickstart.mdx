## Creating your first Project

A project in Karada is the top-level container for all your deployments, integrations, and usage tracking.

### 1. Create a Project

To get started, navigate to your dashboard and click the **Create** button. You can choose to start from a GitHub repository, or create a Provider project (like Auto-MCP).

### 2. View your Dashboard

Once created, you'll be taken to your Project Dashboard. Here you can view your real-time usage metrics, manage team access, and configure project-specific settings.

## Generating an MCP Server

Karada's most powerful feature is **Auto-MCP**, which turns any API into AI-ready tools.

<Steps>
  <Step title='Navigate to Auto-MCP'>
    From your dashboard, click **Create** and select the **Auto-MCP** template.
  </Step>
  <Step title='Provide an OpenAPI Spec'>
    Paste the URL to any valid OpenAPI (`.json` or `.yaml`) specification file
    (e.g., `https://api.example.com/openapi.json`).
  </Step>
  <Step title='Review & Generate'>
    Karada will automatically parse the spec and show you the detected tools.
    Click **Generate** to receive a fully-functioning TypeScript Model Context
    Protocol (MCP) server that proxies requests to your API.
  </Step>
</Steps>
